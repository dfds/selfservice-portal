## Display as `flex` instead of `inline-flex`

It can be useful (eg. in tables) to display the `Badge` as a non-inline element.
Set `inline` to `false` for this behavior.

<Preview>
  <Story name="Inside table">
    <table
      css={css`
        td {
          border: 1px solid #aaa;
          padding: 10px;
        }
      `}
    >
      <tbody>
        <tr>
          <td>
            <Badge flexBoxProps={{ inline: false }}>Custom color</Badge>
          </td>
          <td>Some more content in this next cell</td>
        </tr>
      </tbody>
    </table>
  </Story>
</Preview>
