# Chip

Chips allow users to enter information, make selections, filter content, or trigger actions. For example, chips can be used under a search input to display temporary filters the user has selected from search results.

## Props

| Name        |  Values                            |  Description                                                             | Required |
| ----------- | ---------------------------------- | ------------------------------------------------------------------------ | -------- |
| children    | string                             | Text value to be displayed in the chip.                                  | X        |
| dismissable | boolean                            | Control whether chip can be dismissed. onDismiss prop required, if used. |          |
| onDismiss   | (event: InteractionEvent) => void; | Specifies what happens when "dismiss" icon is pressed, if dismissable    |          |
| disabled    | boolean                            | Disabled visual appearance and interaction disabled.                     |          |
| seleactable | boolean                            | Control whether chip is selectable. onToggle prop required, if used.     |          |
| onToggle    | (event: InteractionEvent) => void; | Specifies what happens when chip is toggled, if selectable.              |          |
| className   | string                             | Allows adding a custom css class.                                        |          |

## Usage

Import the component:

```jsx
import { Chip, ChipContainer } from '@/components/dfds-ui/react-components'
```

Use the component:

export const Template = ({ text, ...args }) => <Chip {...args}>{text}</Chip>

## Container

In most cases, there will be a need to display multiple `Chip` alongside one another. `ChipContainer` can be used for this purpose as follows:
