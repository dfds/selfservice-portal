# Card

The `Card` component can be used to present information to the users. A card can be built using some standard components, that are described in this documentation.
Every component inside a card can be placed in any order, with the exception of the `CardMedia` component.

## Props

<Props of={Card} />

# CardMedia

`CardMedia` component can be used inside a card to present an image or some other media.

## Props

<Props of={CardMedia} />

# CardPriceTag

`CardPriceTag` component can be used inside the `CardMedia` component to display a price tag inside the card.
This `CardPriceTag` is optional, and can be replaced by any other component.

## Props

<Props of={CardPriceTag} />

# CardTitle

The `CardTitle` component can be used inside a card to set a header/title.

## Props

<Props of={CardTitle} />

# CardContent

The `CardContent` component can be used inside a card display content such as body text.

## Props

<Props of={CardContent} />

# CardActions

`CardActions` component can be used inside a card to display any actions (Links, Buttons) that the user can click on.

# Container surface

The `Container` component has a `surface` property that changes the background color.

## Usage

Import the component:

```jsx
import { Card, CardMedia, CardTitle, CardContent, CardActions, CardPriceTag } from '@/components/dfds-ui/react-components'
```

Use the component:
