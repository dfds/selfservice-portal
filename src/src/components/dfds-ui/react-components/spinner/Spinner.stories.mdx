The Spinner supports different variation which can be specified using the `color` prop.

## Light color

The `light` color creates a white variation of the spinner is intended for situations where the spinner needs to be
displayed over a dark surface.

<Preview
  css={css`
    background-color: #002b45;
  `}
>

  </Story>
</Preview>

## Styling

The Spinner has a `className` (and `css` if using `emotion`) prop which allows for custom styling.

### Size

In the following example the size of the Spinner is changed.

<Preview>
  <Story name="Size">

    />
  </Story>
</Preview>

### Centered

In this example the Spinner is styled so that it appears centered in a positioned container.
