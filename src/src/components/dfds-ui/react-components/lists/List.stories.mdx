# Lists

Create lists using the `ListItem` component.

```jsx
import { ListItem, ListText, ListTextGroup, ListIcon, ListDivider } from '@/components/dfds-ui/react-components'
```

## `ListItem`

`ListItem` is a single item in a list and a container of `ListText`, `ListTextGroup` and `ListIcon` components.

Set `multiline` if the item contains more than a single line of text.

Use `clickable` to apply interactive styles (eg. hover and tab navigation). To handle an interaction either use the
`onClick` callback or provide your own component through the `as` prop.

Setting `divider` will display a border underneath the item.

## `ListTextGroup`

Use `ListTextGroup` to wrap `ListText` components to create list items with multiple lines of text. Make sure to set
`multiline` on the parent `ListItem`.

## `ListDivider`

Use `ListDivider` if you want "indent" or "margins" styled dividers between items. Setting the `divider` prop on
`ListItem` will only render the default full length border.

<Preview width="320px" gray>
  <Story name="ListDivider">
    {() => {
      return (
        <React.Fragment>
          <ListItem clickable as="a" href="#">
            <ListText styledAs="label">Default styling</ListText>
          </ListItem>
          <ListDivider />
          <ListItem clickable as="a" href="#">
            <ListText styledAs="label">Margins</ListText>
          </ListItem>

            <ListText styledAs="label">Indent</ListText>
          </ListItem>
