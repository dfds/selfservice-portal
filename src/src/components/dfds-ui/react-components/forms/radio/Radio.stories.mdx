# Radio

Radio allow users to select an option from a list of two or more options that are mutually exclusive.

## Props

| Name      |  Values                             |  Description                                                        | Required |
| --------- | ----------------------------------- | ------------------------------------------------------------------- | -------- |
| name      | string                              | Text value to be displayed as label for radio button.               | X        |
| value     | string                              | Keeps track of radio buttons                                        | X        |
| className | string                              | Allows adding a custom css class.                                   |          |
| checked   | boolean                             | Specifies what happens when radio button is checked, if selectable. |          |
| error     | boolean                             | Specifies what happens if there is an error with selection.         |          |
| disabled  | boolean                             | Disabled visual appearance and interaction disabled                 |          |
| onChange  | (event: InteractionEvent ) => void; | Specifies what happens when the value of radio button is changed    |          |

## Usage
