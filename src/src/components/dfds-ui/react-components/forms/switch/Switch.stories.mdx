# Switch

Switches toggle a state of a property between on and off. The impact of change should be immediate without needing an extra action (such as submitting a form).

## Props

| Name      |  Values           |  Description                                                                                                  | Required |
| --------- | ----------------- | ------------------------------------------------------------------------------------------------------------- | -------- |
| name      | string            | Name of the input element.                                                                                    |          |
| checked   |  boolean          | State of the switch                                                                                           |          |
|  onChange |  () => void       |  Callback to fire when the state of the input changes                                                         |          |
|  disabled |  boolean          |  Controls whether the input is disabled                                                                       |          |
|  right    |  boolean          |  Controls whether input label should be displayed on the right of the input, displayed on the left by default |          |
|  size     |  small \|  medium |  Visual size of the switch                                                                                    |          |
|  children |  string           |  Label of the input                                                                                           |          |
