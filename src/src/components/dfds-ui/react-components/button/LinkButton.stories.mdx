# Link button

Link buttons allow users to navigate from one page to another.

## Props

| Name   |  Values |  Description                               | Required |
| ------ | ------- | ------------------------------------------ | -------- |
| href   | string  | Sets the URL of the page the link goes to. |          |
| target | string  | Specifies where to open linked page.       |          |

## Usage

```jsx
import { LinkButton } from '@/components/dfds-ui/react-components'
```
