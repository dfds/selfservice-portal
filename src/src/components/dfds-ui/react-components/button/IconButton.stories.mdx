# IconButton

Display a button with only an icon.

For accessibility it requires the consumer to provide a value for the \`ariaLabel\` prop.

## Usage

Import the component and an icon you wish to use:

```jsx
import { IconButton } from '@/components/dfds-ui/react-components'
import { Account } from '@/components/dfds-ui/icons/system'
```

## Custom overlay options

You can override the `OverlayOptions` with this prop. In this example even with black icon the overlay color will be the
`lightBlueOverlayOptions` variant.

<Preview>
  <Story name="Custom overlay options">
    <IconButton
      icon={Account}
      ariaLabel="Account"
      color="black"
      overlayOptions={theme.states.overlayOptions.lightBlueOverlayOptions}
    />
  </Story>
</Preview>

## Custom content

The `icon` prop takes a `React.ElementType` so you can provide custom content to be displayed.

In the following example the `BadgeIcon` component is used as custom content.

```jsx
import { BadgeIcon } from '@/components/dfds-ui/react-components'
```

<Preview>
  <Story name="Custom content">
    {() => {
      return (
        <Fragment>
          <IconButton icon={() => <BadgeIcon letter="J" />} tooltip="John Doe" />
          <IconButton
            size="large"
            css={css`
              color: red;
              & text {
                fill: yellow;
                font-weight: bold;
              }
            `}
            icon={() => <BadgeIcon letter="S" />}
            tooltip="Superman"
          />
        </Fragment>
      )
    }}
  </Story>
</Preview>
