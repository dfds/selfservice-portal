# Button stack

Button stack is a container allowing to display multiple adjacent buttons in a container.

## Props

| Name         |  Values                  |  Description                          | Required |
| ------------ | ------------------------ | ------------------------------------- | -------- | --- |
| align        | left \| right \| center  | Align items to the right of the stack |          |
|  orientation | vertical \| horizontal   | Determine orientation of items        |          |
| reverse      | boolean                  | Reverses order of the items           |          |     |

## Usage

```jsx
import { ButtonStack } from '@/components/dfds-ui/react-components'
```
