# Hero

The hero component can be used to present prominent information about the website or any page.
It is responsive in accordance with the Hydro theme.

## Props

| Name       |  Values |  Description                                    | Required |
| ---------- | ------- | ----------------------------------------------- | -------- |
| title      | string  | Title of the hero.                              | X        |
| headline   | string  | Headline describing the hero.                   | X        |
| headlineAs | type    | If true, headline will be H1, else div.         |          |
| imageSrc   | string  | Path to the image shown in the hero.            | X        |
| color      | string  | Set the color of the background and the stripes |          |

## Usage

Import the component:

```jsx
import Hero from '@/components/dfds-ui/react-components'
```

Use the component:
