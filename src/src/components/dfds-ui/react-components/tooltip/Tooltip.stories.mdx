## Hover

Tooltip is shown when button (child) is hovered

  <Story name="Default behaviour">

## Show tooltip when child is clicked

Tooltip is shown when button (child) is clicked and not shown when button is clicked again or when clicked outside tooltip

<Preview>
  <Story name="Click behavior">
    {() => {
      const ToolTipExample = () => {
        const [show, setShow] = React.useState(false)
        return (
          <Tooltip
            content="This is tooltip text"
            tabIndex={-1}
            placement="left"
            visible={show}
            onClickOutside={() => setShow(false)}
          >
            <Button onClick={() => setShow(!show)}>Click me</Button>
          </Tooltip>
        )
      }
      return <ToolTipExample />
    }}
  </Story>
</Preview>
