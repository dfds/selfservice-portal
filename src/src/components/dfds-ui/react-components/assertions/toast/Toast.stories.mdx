## Default Toaster

Import the Toaster component and useToaster hook:

```jsx
import { Toaster } from '@/components/dfds-ui/react-components'
```

Use Toaster Provider component to apply the context to your app:

```jsx
<Toaster>
  <App />
</Toaster>
```

Import useToaster hook to get the context:

```jsx
import { useToaster } from '@/components/dfds-ui/react-components'

const App = () => {
  // get addToast and Toaster state from useToaster hook
  const { addToast, state } = useToaster()
  // invoke addToast function by passing ToastProps to open the Toaster
  return <div onClick={() => addToast({ children: 'Some kind of message' })}>Trigger Me!</div>
}
```

## Long text

While it is possible to display long form text in the Toast, it is not encouraged.
Ideally, toast should have maximum two lines of text:

<Preview>
  <Story name="Long text">
    <Toast>
      Please do not do this! If text has to be this long, it is probably better to use the Banner component.
    </Toast>
  </Story>
</Preview>
