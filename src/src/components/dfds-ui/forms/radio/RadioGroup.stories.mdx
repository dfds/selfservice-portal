# RadioGroup

You can group a list of `Radio` components using the `RadioGroup` component.

```jsx
import { Radio, RadioGroup } from '@/components/dfds-ui/forms'
```

<Props of={RadioGroup} />

## Sizes

Use the `visualSize` prop to specify the size variant.

<Preview width="320px" gray>
  <Story name="Sizes">

  </Story>

## Direction

Use the `column` prop to specify the direction variant.
Default is `true`. Set it to false if you want your radios in a row.

<Preview width="320px" gray>
  <Story name="Directions">
    <form>
      <RadioGroup label="Column by default">
        <Radio name="columnRadio" label="Radio" />
        <Radio name="columnRadio" label="Radio" />
        <Radio name="columnRadio" label="Radio" />
      </RadioGroup>
      <RadioGroup label="Row" column={false}>
        <Radio name="rowRadio" label="Radio" />
        <Radio name="rowRadio" label="Radio" />
        <Radio name="rowRadio" label="Radio" />
      </RadioGroup>
    </form>
  </Story>
</Preview>
