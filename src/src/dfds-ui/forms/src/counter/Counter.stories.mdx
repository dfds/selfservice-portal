# Counter

The `Counter` component can be used to control a number that has a minimum and a maximum with minus and plus buttons.

## Usage

```jsx
import { Counter } from '@/dfds-ui/forms/src'
```

## Basic usage

<Preview>

<Props story="Basic usage" />

Although `minValue` and `maxValue` disable the button appropiately, you can further override this behaviour with `disableMin` and `disableMax`.
