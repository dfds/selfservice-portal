## Limited locales

You can limit the number of locales you want to show in the `LocaleSelector`.

<Preview>
  <Story name="Limited locales">
    {() => {
      const [locale, setLocale] = useState('tr-TR')
      return (
        <LocaleSelector currentLocale={locale} locales={['tr-TR', 'en-GB']} onChange={(locale) => setLocale(locale)} />
      )
    }}
  </Story>
</Preview>

## Disabled LocaleSelector

`LocaleSelector` can also be disabled.

<Preview>
  <Story name="Disabled LocaleSelector">
    {() => {
      const [locale, setLocale] = useState('tr-TR')
      return (
        <LocaleSelector
          currentLocale={locale}
          locales={['tr-TR', 'en-GB']}
          onChange={(locale) => setLocale(locale)}
          disabled
        />
      )
    }}
  </Story>
</Preview>
