# Swiper

The `Swiper` component is a container of `SwiperCard`s used in displaying info that the users can select.

## Props

<Props of={Swiper} />

# SwiperCard

The `SwiperCard` component is a wrapper used in displaying info.

<Props of={SwiperCard} />

## Usage

Import the component:

```jsx
import { Swiper, SwiperCard } from '@/dfds-ui/experiences'
```
