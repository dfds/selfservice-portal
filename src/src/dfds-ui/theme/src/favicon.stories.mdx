# DFDS favicon

## Snippet

Once the assets been copied, insert the following snippet to the `head` section of the page.

```html
<link rel="mask-icon" href="/safari-pinned-tab.svg" color="#002b45" />
<link rel="alternate icon" type="image/png" size="16x16" href="/favicon-16x16.png" />
<link rel="alternate icon" type="image/png" size="32x32" href="/favicon-32x32.png" />
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<link rel="manifest" href="/site.webmanifest" />
<meta name="theme-color" content="#002b45" />
```
