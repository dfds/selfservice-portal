# Checkbox

Checkbox allows user to select one or more options of a limited number of choices.

## Props

| Name          |  Values                             |  Description                                                     | Required |
| ------------- | ----------------------------------- | ---------------------------------------------------------------- | -------- |
| name          | string                              | Text value to be displayed as label for checkbox.                |          |
| value         | string                              | The value of the component.                                      |          |
| className     | string                              | Allows adding a custom css class.                                |          |
| indeterminate | boolean                             | Specifies what happens if one or two sub-options are checked.    |          |
| checked       | boolean                             | Specifies what happens when checkbox is selected, if selectable. |          |
| leadingLabel  | boolean                             | Sets label at leading position.                                  |          |
| error         | boolean                             | Specifies what happens if there is an error with selection.      |          |
| disabled      | boolean                             | Disabled visual appearance and interaction disabled.             |          |
| size          | string                              | Sets the size of checkbox component: 'small' or 'medium'.        |          |
| htmlSize      | number                              | Specifies the number of visible options.                         |          |
| onChange      | (event: InteractionEvent ) => void; | Specifies what happens when the value of checkbox is changed.    |          |

## Usage

```jsx
import { Checkbox } from '@/dfds-ui/react-components/src'
```
