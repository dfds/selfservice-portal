# Input composition

`InputComposition`, `InputControl`, `InputIcon` and `InputAddon` are low level components used internally to
build higher level components like `TextField` etc.

## Usage

```jsx
import { InputComposition, InputControl, InputIcon, InputAddon } from '@/dfds-ui/react-components/src'
```

## Props

<Props components={{ InputComposition, InputControl, InputAddon, InputIcon }} />
