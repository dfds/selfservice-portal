# Banner

Gives information or feedback to the user.
Supports adding two action buttons the user can interact with inside the Banner.

## Usage

```jsx
import { Banner, BannerAction, BannerHeadline, BannerParagraph } from '@/dfds-ui/react-components/src'
```
