# Input Composition

Compose custom `Input` components by composing `InputBase`, `InputElement`, `InputIcon` and `InputAddon`.
These are used internally to build the `Input` component.

## Usage

```jsx
import { InputComposition, InputElement, InputIcon, InputAddon } from '@/dfds-ui/react-components/src'
```
