# Button

Buttons allow users to take actions, and make choices, with a single tap.

## Props

| Name         |  Values            |  Description                                                   | Required |
| ------------ | ------------------ | -------------------------------------------------------------- | -------- | --- |
| variation    | string             | Sets the variation of button.                                  |          |
| size         | string             | Sets the size of the button component 'small' or 'medium'.     |          |
| disabled     | boolean            | Disabled visual appearance and interaction disabled.           |          |
| icon         | component          | Allows adding a Icon component.                                |          |
| iconAlign    | string             | Specifies the alignment of the icon 'left' or 'right'          |          |
|  iconSpacing |  string            | Allows defining custom spacing betwene the text and the icon   |          |
|  as          |  React.ElementType | Sets tag of the component                                      |          |
| href         | string             | Url the button navigates to, when rendered as "a"              |          |
| submitting   | boolean            | Shows spinner, has disabled appearance and is not interactable |          |     |

## Usage

```jsx
import { Button } from '@/dfds-ui/react-components/src'
```
