# Container

The `Container` centers the content horizontally.

## Component props

<Props components={{ Container, Column }} />

## Usage

```jsx
import { Container, Column } from '@/dfds-ui/react-components/src'
```

export const Box = (props) => {
  return (
    <div
      css={css`
        height: 100px;
        background-color: lightgrey;
        display: flex;
        font-size: 16px;
        font-family: 'DFDS';
        align-items: center;
        justify-content: center;
      `}
      {...props}
    />
  )
}

## `Container xx-large, x-large and large`

The `Container` component on xx-large, x-large and large viewport supports up to 12 column on each row.

Size above 1360px for xx-large. Content remains at max-width 1200px, side margins grow.

Size between 1280px - 1359px for x-large. Content remains at max width 1200px, side margins grow.

Size between 1025px - 1279px for large. Content starts to shrink below 1200px with fixed side margins.

## `Container medium`

The `Container` component on a medium viewport only supports up to 2 columns per row.

Size down to 768px - 1024px to see medium viewport.

## `Container small`

In small viewports, columns inside `Container` will stak on top of each other.

Size down to 0px - 767px to see mobile viewport.

## `Column`

The `Column` component supports sizing props to adjust the sizing of each column.

<Preview>
  <Story name="Column">

      </Column>

  </Story>
</Preview>

## `Column without sizing props`

Columns without sizing will automatically take up the remaining size of the grid.
