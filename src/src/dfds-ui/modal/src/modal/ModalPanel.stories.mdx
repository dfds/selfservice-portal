# Modal Panel

Component to normally display a panel in full screen on mobile devices.

Support a fixed header and footer with a scrollable content area in the middle.

## Props

<Props of={ModalPanel} />

## Usage

Install with npm:

```jsx
npm install --save @/dfds-ui/modal/src
```

Install with yarn:

```jsx
yarn add @/dfds-ui/modal/src
```
